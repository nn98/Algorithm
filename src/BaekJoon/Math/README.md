>#### 수학 쉽지않다

> {n - 순열의 합(l == 4 일때 1+2+3+4)} / l 이 시작값<br />
> 시작값이 정수로 나누어 떨어지면 그값부터 순열 출력<br />
> 아니면 자릿수(순열 길이) 1 늘리고 반복<br />
> 반복하다 자릿수 100 넘어가면 못만드니 -1
